# 21_cub3D

The goal was to make a dynamic view inside a maze, in which the player should find a way.

Regimes.

To play:
- make && ./cub3D data.cub
- make bonus && ./cub3D data.cub
- make bonus && ./cub3D data_neon.cub
- make cabaret && ./cub3D data_cabaret.cub
